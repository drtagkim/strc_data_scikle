# strc_data_scikle
스마트관광연구소 툴킷
## 설치 방법
* Chrome 설치
* 주소창에 chrome://extensions
* 오른쪽 상단 "개발자 모드" on
* 다운로드 받은 프로젝트를 압축해제
* "압축해제한 확장 프로그램을 로드합니다." 선택하여 압축해제한 폴더 지정

## 크롬 종료 전에
* chrome://extensions
* 개발자 모드로 된 확장프로그램은 사용하지 않을 때 사용을 off로 해줘야 함
* 다음 사용할 때는 다시 chrome://extensions에 들어와서 on하면 됨

### 개발
수원대학교 경영학부 김태경 교수
